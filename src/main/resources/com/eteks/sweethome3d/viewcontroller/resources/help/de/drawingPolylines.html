     Polylinien und Kurven zeichnen
 

Um Polylinien und Kurven zu zeichnen, müssen Sie zuerst den Menüeintrag Plan > Polylinien erzeugen oder das Werkzeug Polylinien erzeugen auswählen.

 Werkzeug Polylinien erzeugen

Sie können eine neue Polylinie erzeugen, indem Sie im Plan den Startpunkt der neuen Polylinie anklicken. Klicken Sie dann jeden Eckpunkt der Polylinie an, bis Sie ihren Endpunkt doppelt anklicken oder die Escape-Taste drücken. Wenn der abschließende Doppelklick am selben Punkt wie der Startpunkt stattfindet, wird der Pfad der Polylinie geschlossen und ein Polygon wird erzeugt. Wenn Sie die Strg-Taste (oder unter Mac OS X die Alt-Taste) drücken, während Sie das erste Liniensegment zeichnen, wird eine Kurve erzeugt, sofern die Polylinie aus mindestens drei Punkten besteht.
Polylinien und Kurven erscheinen nur im Wohnungsplan. In der 3D-Ansicht sind sie unsichtbar.

Verschiedene Polylinien, Kurven und Polygone

Um das Zeichnen von Polylinien zu beenden, wählen Sie Plan > Auswählen oder wählen Sie das Auswählen-Werkzeug (oder ein anderes Werkzeug).

 Auswählen-Werkzeug

•

Benutzen Sie ToolTips und die Vergrößerungsfunktion, um Polylinien exakt zu zeichnen. Wenn Sie die Eingabetaste drücken, während der Wohnungsplan fokussiert ist, werden die im ToolTip angezeigten Werte editierbar, sodass Sie per Tastatur die Länge und den Winkel des aktuellen Segments eingeben können. (Falls Sie den ersten Punkt der Polylinie noch nicht festgelegt haben, werden im ToolTip die Koordinaten jenes Punktes angezeigt.) Benutzen Sie die Pfeiltasten aufwärts und abwärts oder die Tabulatortaste, um einen anderen Wert zu bearbeiten, und drücken Sie erneut die Eingabetaste, um die eingegebenen Werte zu bestätigen. Wenn Sie die Polylinie mit der Maus weiterzeichnen möchten, bewegen Sie den Mauszeiger einfach über den Plan.

•

Der Winkel des gezeichneten Segments ist standardmäßig ein Vielfaches von 15°. Um zu verhindern, dass der Winkel auf derartige Werte einrastet, schalten Sie die Funktion im Bereich Einstellungen bearbeiten aus. Alternativ können Sie dazu auch beim Zeichnen die Alt-Taste in Windows, die Cmd-Taste unter Mac OS X oder die Umschalttaste und die Alt-Taste unter Linux gedrückt halten.

•

Wie in der oben gezeigten Grafik an der letzten Form zu sehen ist, wird eine Polylinienkurve so berechnet, dass ein geschlossenes Quadrat aus vier Punkten eine kreisförmige Kurve ergibt.

•

Wurde eine Polylinie erstellt, können Sie ihre Position, ihre Farbe und ihren Stil ändern. Im Bearbeitungsbereich können Sie eine Polylinie auch noch nachträglich in eine Kurve umwandeln.

•

Wählen Sie Plan > Vergrößern oder Plan > Verkleinern, um den Maßstab des Plans zu ändern. Bewegen Sie alternativ das Mausrad bei gedrückter Strg-Taste (beziehungsweise Cmd-Taste unter Mac OS X) auf oder ab.

•

Während das Werkzeug Polylinien erstellen ausgewählt ist, sind einige andere Werkzeuge und Menüs inaktiv. Möchten Sie diese Werkzeuge benutzen, aktivieren Sie wieder das Auswählen-Werkzeug.

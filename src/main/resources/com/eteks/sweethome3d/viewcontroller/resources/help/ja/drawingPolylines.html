     多角線・曲線を描く
 

多角線、曲線を描くにはまず、プラン > 多角線、曲線の作成 をメニューか選ぶか、又は 多角線・曲線作成 ツール.

 多角線・曲線の作成 ツール

プラン上で新たに多角線を作成するには、線の始めの位置をクリックして、各々の曲がるポイントでクリックをして、線の最後でダブルクリック又はEscapeキーを押します。線の最後が始めと重なり、そこでダブルクリックをすると多角線が多角形になります。 Ctrl キー (又は Mac OS Xの場合はalt キー)を押しながら最低３点をクリックして最初のセグメントを描くと曲線ができます。
多角線・曲線はプラン上だけて表示されて3Dビューでは表示されません。


いろいろな多角線、曲線、多角形

多角線の作成終了はプラン> 選択 又は 選択 ツール (又は他のツール)を選択します。

 選択 ツール

•

正確に多角線を描くためにツールチップとズームインが使えます。プランでフォーカスがある時、enterキーを押すとツールチップが表示されて、選択されているセグメントの長さ、角度が入力、変更が可能になります。 （多角線の先端が選択されていない場合はその座標がツールチップに表示されます。) 上方矢印, 下方矢印 又は tabキーで値を更新し、enterキーを再度押して、更新確認をします。 多角線・曲線をマウスを使って再度描きたい場合はマウスカーソルをプラン移行するだけでできます。

•

線の角度は初期化でx15°です。この磁性をキャンセルしたい場合は設定 画面に移行するか、もしくは　WindowsはAltキー、Mac OS Xはcmdキー LinuxはShift + Altは押し続けます。

•

上のイメージの最後の形で解るように、多角線に４つのポイントが存在しかつ両端を結びつけることで円が作成できます

•

多角線が作成された後、更新 でその位置、色、スタイルを変えることができます。更新画面で多角線を曲線にも変えることが可能です。

•

プラン > ズームイン又は、プラン > ズームアウトで縮尺を変えることができます。ctrlキー (Mac OS Xはcmdキー)を押しながらマウスウィールを前後に動かすことで縮尺を変えることもできます。

•

多角線・曲線の作成ツールを選択したら、他のツール及びメニューが無効になることがあります。有効にするには選択 ツールを再度選びます。

Руководство пользователя Sweet Home 3D > Создание дома, стен и мебели
     Печать плана дома
 

Для печати плана дома следует выбрать Файл > Печать....
Это меню вызывает системный диалог печати, где можно выбрать принтер, какие страницы печатать и качество печати...

По умолчанию Sweet Home 3D печатает список мебели, план и текущий 3х мерный вид, используя бумагу, ориентацию страницы и отступы, назначенные по умолчанию.
Если вы хотите это изменить, выберите Файл > Параметры страницы....

В настройках страницы Вы можете изменить размер страницы и ориентацию, кликнув на кнопку Формат страницы. Вы можете выбрать возможность печати списка мебели, плана и вида 3D. Если вас не устраивает автоматическое масштабирование плана к размерам листа бумаги, вы можете выбрать свой масштаб в поле Масштаб.
Так же вы можете задать верхний и нижний колонтитулы, которые будут печататься на каждой странице. Текст помещаемый в колонтитулы может содержать переменные, которые будут заменены их значениями на листе бумаги. На текущий момент доступно 7 переменных:

  • $pageNumber номер страницы,
  • $pageCount общее количество страниц,
  • $planScale масштаб (автоматически расчитанный, или введённый в поле Масштаб),
  • $date текущая дата,
  • $time текущее время,
  • $name название дома,
  • $file имя файла и путь к нему.

Чтобы не ошибиться в названии переменных, воспользуйтесь кнопками Переменные, расположенными под полями колонтитулов. Символ $ зарезервирован для определения переменных, если необходимо напечатать $, то, в нужном месте, введите $$.

Для предварительного просмотра, выберите Файл > Предварительный просмотр....

В панели предварительного просмотра, вы увидите, как будет выглядеть страница на печати. Для смены страницы, нажмите на стрелки сверху или нажмите стрелки на клавиатуре.

•

Печать под Linux может не работать. Вместо этого вы можете сделать Файл > Сохранить в PDF... и распечатать полученный файл.

•

Печать 3х мерного вида может быть недоступна в виду ограничений вашей системы. В этом случае, галочка Печатать трёхмерный вид будет недоступна.

•

Если печатается дом, содержащий больше одного уровня, то список мебели будет содержать только мебель, видимую на текущем уровне, за исключением случая, когда вам понадобится напечатать только список мебели, тогда будет напечатан полный список мебели. Печать плана и 3х мерного вида напечатают текущий уровень. Для печати плана или 3х мерного вида нескольких уровней, выберете печать каждого уровня по отдельности.

•

Вид 3D печатается с разрешением 150 dpi.

•

При необходимости, отобразить масштаб на плане, вы можнте нарисовать размерную линию стандартной длины (например 1 метр).

•

Настройки страницы сохраняются вместе с планом.

•

Панель предварительного просмотра, при необходимости, можно растягивать.

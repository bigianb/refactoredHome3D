Руководство пользователя Sweet Home 3D > Создание домов, стен и мебели
     Сохранить и открыть дом
 

Для сохранения дома, выберите пункт меню Файл > Сохранить, или, Файл > Сохранить как....
Каждый файл Sweet Home 3D имеет расширение .sh3d. Если Вы не ввели расширение, то Sweet Home 3D автоматически добавит его к файлу.
После сохранения, имя файла будет отображаться в заголовке окна и в меню Файл > Открыть ране сохраненные файлы.

Вы можете также:

  • Экспортировать список мебели в CSV формат (с табуляцией, в качестве разделителей), выбрав пункт меню Мебель > Экспорт в формате CSV.
  • Экспортировать домашний план в файл формата SVG (Масштабируемая Векторная Графика), выбрав пункт меню План > Экспорт в формате SVG...,
  • Создать изображение 3х мерного просмотра в формате PNG, выбрав пункт меню Вид 3D > Создать фото...,
  • Создать видео 3х мерного просмотра в формате Quicktime MOV, выбрав меню Вид 3D > Создать видео...,
  • Экспоровать 3х мерное представление дома в формате OBJ, выбрав пункт меню Вид 3D > Экспорт в формате OBJ.... Эта особенность позволяет Вам создать файл OBJ, который опишет все объекты трехмерного представления, вместе с файлом MTL и изображениями текстур, которые будут созданы в той же самой папке.

Чтобы открыть существующий файл с планом, можно дважды кликнуть на нем в менеджере файлов, выбрать в меню Файл > Открыть... или открыть из списка ранее открываемых файлов в меню Файл > Открыть ранее сохраненные....

•

В файле Sweet Home 3D сохраняются так же фоновое изображение которое, вы импортировали, параметры 3х мерного просмотра, и все 3х мерные модели мебели добавленной в ваш дом. Это позволяет обмениваться файлами между пользователями, у которых отсутствуют необходимые модели в каталоге мебели.

•

Выберите Файл > Сохранить и сжать, чтобы сохранить Ваш дом в сжатом формате, полностью совместимом с Sweet Home 3D форматом. Эта операция занимает больше времени, но позволяет создать меньший по размеру файл.

•

Выберите Файл > Открыть недавние > Очистить меню, если хотите очистить список ранее открытых файлов.

Руководство пользователя Sweet Home 3D
  Подсказки
 

Здесь несколько подсказок, помогающих сделать работу в Sweet Home 3D быстрой и удобной.

Основное

•

Используйте разделители окон между каталогом фурнитуры, списком фурнитуры, планом, видом 3D для изменения размера окон.

•

Во время проверки всех доступных возможностей Sweet Home 3D, не забывайте, что есть операция отката, которая действует везде, кроме операций импорта мебели в каталог.

Каталог мебели

•

Для добавления более, чем одного объекта на план за раз, выберите нужные объекты в каталоге.

•

Для открытия категории каталога, дважды кликните на ней.

•

Для изменения импортированной мебели, дважды кликните на ней.

•

Для работы с большим каталогом мебели и фурнитуры поможет настройка Список с поиском. Окно выбора категорий и текстовая строка поиска помогут найти необходимые предметы.

Список мебели, используемой в доме

•

Для сортировки списка мебели, кликните на заголовок колонки по которой хотите сортировать. При первом клике будет сортировка по возрастанию, при повторном - по убыванию, при третьем клике реализуется сортировка по порядку добавления в список.

•

Можно изменить порядок колонок, просто перетаскивая заголовки в нужном порядке.

•

Для поиска местоположения предмета мебели на плане, выделите ее в списке мебели, используемой в доме.

•

Для изменения предмета, кликните на нём дважды на плане.

•

Для изменения более чем одного предмета, выделите их на плане или списке, и выберите меню Мебель > Изменить....

•

Если возникает необходимость объединить предметы, выделите их и сгруппируйте Мебель > Группа. Такая группа предметов может быть перемещена, изменена или повёрнута, как единый элемент.

План дома

•

Если у Вас есть отсканированный план дома, Вы можете его "подложить" под план с помощью импорта фонового изображения в план перед работой. Это поможет быстрее и точнее нарисовать стены.

•

Для точных корректировок используйте масштабирование.

•

Если хотите сделать стены разноцветными, то составьте их из отдельных сегментов с помощью создания или выберите пункт меню План > Разделить стену.

•

Для создания стен с наклоном, выберите различную высоту в начале и конце стены.

•

При выборе Редактировать > Копировать или Редактировать > Вырезать, выбранные объекты в плане будут скопированы в буфер обмена, так же, как картинки, и вы сможете вставить их в другие приложения. Под Linux это может не работть. Если хотите сохранить план в векторном формате, выберите в меню План > Экспорт в формате SVG....

•

Используйте стрелки клавиатуры для точного позиционирования мебели.

•

Зафиксируйте основной план для выбора и перемещения мебели, чтобы случайно не сдвинуть другие объекты на плане.

•

При необходимости выбрать объекты, скрытые за другими, выберите их из списка мебели, если этот предмет относится к мебели, или расположите курсор мыши над скрытым объектом и правым кликом мыши откройте контекстно меню и выберите нужный объект вподменю Выбрать объект. Или вы можете выделить прямоугольную область содержащую нужный объект и выбрать его мышкой с зажатой клавишей shift.

•

Если необходимо снять выделение с объекта, из нескольких выделенных, нажмите на нём, удерживая клавишу shift. Если объект спрятан за другим, то вызовите контекстное меню правой кнопкой мыши и выберите нужный объект в подменю Обратить выделение, с нажатой клавишей shift, это подменю заменяет подменю Выбрать объект, когда нажата клавиша shift.

•

Используйте выравнивающие линии для точного расположения объектов.

•

Для перемещения выделенных объектов вдоль горизонтальных или вертикальных выравнивающих линий используйте стрелки клавиатуры, или перетаскивайте их, нажав клавишу shift, после того,как начали перемещать.

•

Для клонирования выбранных объектов на плане, перетаскивайте их, зажав клавишу ctrl (в Mac OS X необходимо зажимать клавишу alt).

•

Притяжение, настроенное в панели настроек, может быть временно изменено, если во время выполнения операции зажать кнопку alt (в Mac OS X необходимо зажимать клавишу cmd).

•

Для облегчения рисования стен и расположения мебели на плане можно изменить точку просмотра и прозрачность стен в 3х мерном просмотре.

•

Для выравнивания мебели выберите необходимые предметы и выберите пункт меню Мебель > Выровнять.... Предметы мебели будут выровнены по первому выбранному предмету.

•

Для перемещения видимой части плана без использования полос прокрутки справа и внизу окна, можно воспользоваться пунктом меню План > Перемещение. Другой вариант активации этого способа перемещения может быть осуществлен нажатием и удерживанием клавиши пробел.

•

Выберите пункт меню План > Увеличить или План > Уменьшить, для изменения масштаба или, при нахождении курсора над планом дома, прокрутите колёсиком мыши при нажатой клавише ctrl (При использовании MacOS X удерживайте cmd).

3х мерный просмотр дома

•

Выбор меню Вид 3D > Показать в другом окне позволит переместить окно 3D вида в любое место экрана. Чтобы вернуть 3х мерное представление в главное окно достаточно выбрать Вид 3D > Показать в главном окне или закрыть окно 3х мерного просмотра.

•

Нажатие клавиши shift увеличивает скорость передвижения мыши и вращения колеса в 3х мерном просмотре.

•

Если вы не видите мебель в какой-нибудь комнате, попробуйте изменить прозрачность стен.

•

Если яркость завышена и все блестит, измените яркость освещения.

•

Если Вы не можете найти положение виртуального посетителя, то покрутите колесо мышки, в 3х мерном просмотре это автоматически выберет и покажет посетителя на плане.

•

Используйте 3D вид > Сохранить вид... для основной точки просмотра, к которой вы всегда можете вернуться, выбрав пункт меню 3D вид > Перейти к виду. Время, дата, и тип линзы, установленные в панели Создать фото, так же сохраняются в текущей точке просмотра. При необходимости выберите 3D вид > Удалить виды..., для удаления ненужных точек просмотра.

•

По умолчанию, изображения создаваемые на двух лучших уровнях качества при помощи панелей создать фото и видео освещаются светильниками, расположенными в центре каждой комнаты и солнцем в дневное время. При необходимости, можно отключить центральное освещение, выключив галочку Добавить потолочные светильники в этих панелях, изменить текущую дату и время, изменить ориентацию дома по сторонам света, добавить новые светильники и отрегулировать их мощность.

•

Вы можете экспортировать 3х мерный вид в OBJ формат и импортировать полученный файл в другие программы, например Art of Illusion или Blender.

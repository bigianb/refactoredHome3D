     繪製折線和曲線
 

　　繪製折線和曲線，你必須選擇 平面圖→建立曲線 功能選單項目或選擇建立曲線工具圖示。

 建立曲線工具

　　您可以通過按一下新的折線，然後在每個點的折線上直到你按兩下在其終結點上或起始點在平面圖中建立新的折線按Escape鍵，就可結束畫折線。建立的多邊形，如果最後按兩下位於起始點上，然後折線的路徑將被結束， Ctrl 鍵或alt 鍵( Mac OS X) 。折線計數至少有 3 個點，如果在繪製第一段時，則會建立一條曲線。
　　折線和曲線僅在該平面圖\中出現，不會顯示在 3D 視圖中。

各種折線、 曲線和多邊形

若要繪製折線的結束，選擇平面→選擇或選選定圖中的對象工具 (或其他工具)。

 選擇工具

•

為了讓您更精確地繪製折線，請使用工具提示和放大。如果你按下Enter鍵，將出現工具提示顯示的值將成為可編輯的數字，讓您可輸入鍵盤數字，長度和當前線段的角度 (如果你沒有跡象表明折線的第一個點，然而，工具提示將顯示該點的座標)。 使用向上箭號鍵、向下方向鍵或tab鍵編輯其他值，再按Enter鍵確認您輸入的值。如果你想要再次使用滑鼠以繪製折線，只需移動到平面圖上既可。

•

繪製的角度在預設情況下是線段的 15 度的倍數。您可以取消這一磁性在首選項窗格 或繪圖時，通過按住在 Windows (Alt鍵)、 Mac OS X ( cmd鍵)、 Linux(Shift+Alt鍵)。

•

如圖所示的最後一個形狀在上面的圓形圖像，然後以這種方式，創建一個封閉的圖形，如果您創建方形 4 個點計算折線的曲線，可以產生的曲線將是一個圓圈。

•

一旦創建了一條折線，你可以更改它的位置、顏色和樣式。之後您還可以在修改窗格中更改折線曲線。

•

選擇平面圖 > 放大或平面圖→縮小更改平面圖比例，或滾動滑鼠輪向前或向後，同時按下Ctrl鍵 (或在 Mac OS X按下cmd鍵)。

•

雖然選擇了創建折線工具，一些工具和功能表被禁用。如果你需要這些工具，再選擇選定圖中的對象的工具。

Sweet Home 3D 說明 → 自訂參數設置、平面圖和 3D 視圖
     自訂 3D 視圖
 

　　您可以選擇 3D 視圖→鳥瞰模式或 3D 視圖→虛擬參觀在兩種 3D 遊覽模式間切換。

鳥瞰模式 虛擬參觀模式

　　當您使用“鳥瞰模式”時，3D 視圖中顯示的是從高處俯瞰家居模型的三維場景。在該模式下按住滑鼠左鍵，左右拖曳滑鼠可使家居模型圍繞其中心處的縱軸旋轉；上下拖曳滑鼠可使家居模型沿著與視圖底部平行的水準軸旋轉；滾動滑鼠滾輪則可縮放 3D 視圖的顯示。
　　當您使用“虛擬參觀模式”時，3D 視圖中顯示的是從一個虛擬參觀者的視角見到的三維場景。在該模式下按住滑鼠左鍵，左右拖曳滑鼠可使虛擬參觀者向左或向右轉身；上下拖曳滑鼠可使虛擬參觀者抬頭或低頭；滾動滑鼠滾輪則可使虛擬參觀者向前走或向後退。
　　在這兩種模式中，使用鍵盤上的左右方向鍵等同於左右拖曳滑鼠，使用 Page Up 或 Page Down 鍵等同於上下拖曳滑鼠，使用上下方向鍵則等同於滾動滑鼠滾輪。

　　使用“虛擬參觀模式”時，平面圖中會顯示出一個虛擬參觀者，其位置和角度會在您操作 3D 視圖時同步變化。虛擬參觀者的身上共有四個標示符：

1.

頭部角度標示符表示您可以拖曳此處使虛擬參觀者抬頭或低頭。

2.

視線標示符表示了虛擬參觀者當前在 3D 視圖中的視角。

3.

身體角度標示符表示您可以拖曳此處使虛擬參觀者向左或向右轉身。

　　當滑鼠指標位於一個訪問者肩上或在其背上時，它將更改為指示您可以拖動，並將這一點來更改的觀視者，它的身體角或其觀視點海拔的頭部角度。同時按下滑鼠按鈕，工具提示顯示已編輯的值。
您也可以從家庭計畫:更改虛擬觀視者的位置，拖動和刪除它，或選擇它，然後按鍵盤上的箭頭。
按兩下計畫中的虛擬觀視者，可以顯示虛擬觀視者中窗選方格，或通過選擇設置的觀視者和先前描述的角度位置3D 視圖→修改虛擬觀視者...功能表項目。

　　此窗選方格還允許更改虛擬觀視者的視野並設置是否應按照當前選定的樓層，使它向上或向下，向所選樓層的虛擬觀視者調整總仰角。

　　最後，這3D 視圖→修改3D視圖... 功能表項目顯示 3D 視圖窗選方格中，您可以更改地面和天空的顏色或紋理、以及光亮度和牆壁(或地面)的透明度。

•

在 3D 視圖中拖曳或滾動滾輪的同時按住 Shift 鍵可以減小三維場景變化的速率。

•

您可以在 3D 視圖中使用 W、S、A 和 D 代替向上、向下、向左和向右的方向鍵。

•

您在 3D 視圖中進行的操作不可撤銷，但 3D 視圖的當前狀態會在您進行您保存檔案的操作時一併存入 Sweet Home 3D 檔案中。

•

僅當 3D 視圖捕獲焦點時您才可以使用鍵盤方向鍵更改虛擬參觀者的頭部和身體角度。當平面圖捕獲焦點時，在虛擬參觀者身上使用方向鍵的結果，與針對其他傢俱一樣，僅僅是移動其位置。

•

上下拖曳的同時按住 Alt 鍵等同於前後滾動滾輪。

•

為防止虛擬觀視者在牆內或非常離家很遠的地方，無法顯示虛擬觀視者，如果你不知道虛擬訪問者所處的位置，在 3D 視圖中用滑鼠指標滾動滑鼠輪，它將會自動顯示虛擬觀視者在繪圖環境中

•

這依照所選擇的樓層去調整眼睛高度選項是預設情況下選中的並且有效，僅當已編輯的房間有多個樓層。如果您要保留預設以樓層為提升高度的視點，就無需取消，若要以高程為視點，您也可以取消該項選點。

•

如果你想始終看到 3D 視圖中的所有樓層，無論所選的樓層是，選擇 3D 視圖→顯示所有樓層 功能項目。去顯示被選取的樓層和所有其他較低的樓層, 選取3D 視圖→顯示被選取的樓層 功能項目。

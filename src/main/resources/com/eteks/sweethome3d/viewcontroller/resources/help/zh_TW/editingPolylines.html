     修改折線及曲線
 

　　選擇他們在平面圖後，您可以用滑鼠編輯折線的位置。
　　要選擇平面圖中的一條折線，請按一下它。若要選擇多個折線或多個物件在平面圖中，選擇在周圍繪製矩形，或 / 和按shift鍵的同時按一下它們。
　　若要移動所選的傢俱，將滑鼠指標放一個選定框內的這些傢俱，然後拖和放它們，或按鍵盤箭頭。

　　當平面圖在選定的一條折線時，在每個點選定的折線點上會出現有點的指標，你可能會移動任何折線的點。

　　當滑鼠指標位於這些折點指標之上時，它將改變這指標，您可以拖放那點移動它。
　　顏色和折線的樣式也可在其窗格中編輯，當出現折點窗格後雙擊它， 或選擇平面圖→修改折線...然後選擇它。

　　在折線窗格中，您可以更改 所選的折線厚度、在繪製箭頭兩端、轉折點樣式、線段的樣式和顏色。最後的轉折點樣式下拉清單中選擇允許繪製一條曲線，而不是線。

•

如果您想要修改的樣式的一整組的折線，選擇這些折線，然後選擇平面圖→修改折線...。從一條折線到一組折線組是不同的屬性的值，將出現在窗格中的轉折線樣式是空白的。 您也可以複製貼上折線樣式到其他折線上，通過複製貼上使用編輯→複製功能表項目，然後選擇編輯→貼上樣式您想要選擇修改的折線。

•

如果你想要禁用啟動的磁性為你繪製，禁用磁性在參數設置窗格中或移動點時，按住Alt鍵在 Windows下、 在Mac OS X 下按cmd鍵，在 Linux 下按 + Alt鍵。

•

如果在平面圖中所選的物件包含的牆壁、房間、傢俱、標註、折線和文字，所有這些物件將使用拖放或鍵盤箭頭移動。如果你按住Ctrl鍵 (或 Mac OS X下， alt鍵)拖拉這些物件時，這些物件將被覆制過去。

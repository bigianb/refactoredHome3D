Sweet Home 3D 說明 → 自訂參數設置、平面圖和 3D 視圖
     自訂牆體
 

　　如果您要自訂家居模型中牆體的位置和長度，您既可以用滑鼠直接在平面圖中操作，也可以使用平面圖→自訂牆體... 功能表項目。
　　如果您想用滑鼠調整牆體，您應當首先在家居平面圖中將其選中。在需要選中的牆體上按一下滑鼠可以選中單堵牆體。如果要選中平面圖中的多堵牆體，請在其周圍拖曳出一個選框，或者在按住 Shift 鍵的同時依次按一下每一堵牆體。
　　如果要移動選中的牆體，請將滑鼠指標移到平面圖中任意一堵已選定的牆體上，然後拖曳到您想要的位置；您也可以使用鍵盤上的方向鍵進行該操作。

　　如果平面圖中只有一堵牆體被選中的話，拖曳被選牆體兩端的標示符可以更改牆體起點和終點的位置。

1.

起點處的標示符表示您可以拖曳此處更改牆體的起點位置。

2.

牆體起始標示符表示牆體的起點。

3.

牆體終止標示符表示牆體的終點。

4.

終點處的標示符表示您可以拖曳此處更改牆體的終點位置。

　　當滑鼠指標位於選中牆體的起點或終點上時，其形狀會發生變化，指示您可以拖曳滑鼠移動該點。當滑鼠按鈕處於按下狀態時，滑鼠指標的旁邊會顯示出標明牆體長度的工具提示。
　　使用滑鼠直接更改的牆體會在平面圖和 3D 視圖中同步呈現。

　　自訂牆體的另一種途徑是使用“自訂牆體”對話方塊，方法是在平面圖中按兩下您想要更改的牆體，或者在選中要更改的牆體後使用平面圖→自訂牆體... 功能表項目。

　　在該對話方塊中，您可以更改牆體起點和終點的座標位置、牆體左右兩側的紋理或顏色，以及其厚度、高度、弧形及圓形的牆，其牆的顏色及光澤。您還可以增加或刪除牆底的踢腳板，通過點擊修改底板按鈕。
　　如果您更改的這堵牆體是尖頂房屋頂樓的山牆，您可以在起點和終點處設置不同的高度，但此時必須確保“梯形牆體”選項按鈕處於選中狀態。
　　您可以根據平面圖中選中牆體上顯示的牆體起始和終止標示符辨別牆體的左側和右側。

　　在牆上窗格中，您可以更改它的開始和結束點、 顏色、 紋理和其左側和右側，用於填補計畫中的圖案，用於繪製 3D 視圖、 它的厚度，其高度和其弧多大程度上其頂部，如果它是一個圓形的牆的顏色光澤的座標。你可能還添加 (或刪除) 踢腳板底部的每一方通過點擊修改主機板按鈕。

•

如果您想要更改一系列牆體的顏色、紋理、厚度或高度，請將它們全部選中，然後在功能表上選擇平面圖→自訂牆體... 。如果這些牆體的某些屬性不盡相同的話，對話方塊中相應的數值框會顯示為空白。如果這些牆體中的某一堵牆體方向不正確的話，請將其選中，然後選擇平面圖→更改牆體方向。

•

如果一系列相鄰的房間位於同一堵牆體的同一側，而您最終要將不同的顏色或紋理應用到位於各個房間的相應部分，那您應當在繪製牆體的過程中就按照多堵牆體來處理。或者，您也可以在完成繪製後使用平面圖→切分牆體功能表項目將一堵牆體分割為兩堵。

•

如果您不想手動計算終點座標的話，您可以使用“起點至終點的長度”數值框。改動該數值的同時，軟體會自動計算出牆體終點的 X 與 Y 座標值，反之亦然。

•

為使您能方便地維持牆體的互相平行，預設情況下用滑鼠更改的牆體的角度總是 15 度的倍數。拖曳滑鼠的同時按住 Shift 鍵可以臨時遮罩該自動吸附的功能，而禁用“參數設置”中的“自動吸附”則可以直接關閉該功能。

•

如果您在平面圖中選中的內容包含有牆體、房間、傢俱、標註和文字，那麼所有這些傢俱會在您使用滑鼠拖曳或鍵盤方向鍵時一起移動。拖曳這些傢俱時按住 Ctrl 鍵（Mac OS X 中為 Alt 鍵）可以進行複製的操作。

•

長度工具提示和“自訂牆體”對話方塊中使用的預設單位取決於作業系統的區域設置，但您仍然可在“參數設置”中將其更改為釐米、米、毫米或英寸。

•

使用平面圖→放大或平面圖→縮小可以更改平面圖的顯示比例。

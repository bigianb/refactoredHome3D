Sweet Home 3D 說明 → 插入背景圖像與傢俱
     插入傢俱
 

　　如果要將現有的 3D 模型插入傢俱目錄或家居平面圖中，您可以在功能表上選擇傢俱→插入傢俱... ，也可按一下“插入外部傢俱”工具列按鈕，或直接將要插入的 3D 模型檔拖放到 Sweet Home 3D 窗口中。

 插入外部傢俱工具

　　該操作會開啟傢俱插入嚮導，您只需按照說明操作即可完成 3D 模型的載入及其相關參數的設置。

1.

將 3D 模型拖放到嚮導視窗的方框內，或按一下“選擇模型”按鈕選擇模型檔。支援的 3D 模型格式有 OBJ、DAE、KMZ 和 3DS，同時您也可以選擇內含支援檔案格式的 ZIP 壓縮包。載入模型後，請按一下“繼續”。

2.

如果模型的朝向不正確，您可以使用視窗內的箭頭按鈕對模型進行定向操作，直至主視圖與模型的正面相符。個別 3D 模型有時可能會因其正反兩面被倒置而被錯誤地渲染。當這種情況發生時，請選中“翻轉法線”核取方塊。
確認模型的渲染沒有問題後，按一下“繼續”。

3.

如有必要的話，您可以更改被插入模型的名稱、大小、懸空高度、顏色、可移動性，等等，或者定義其是否為門或窗。
如果您執行插入模型的操作時螢幕焦點位於傢俱清單或家居平面圖中，您便可以使用“增加到目錄”核取方塊決定是否將該傢俱同時插入傢俱目錄中。
去掉“維持比例”核取方塊內的鉤，您就可以任意設置 3D 模型的寬度、高度與深度，各個數值間互不影響。
如果您在更改 3D 模型的顏色後想將其恢復到默認顏色，請按一下“清除顏色”按鈕。
確認模型的渲染沒有問題後，按一下“繼續”。

4.

用滑鼠將 3D 模型旋轉到您認為合適的角度，以此為基準生成的圖像會在傢俱目錄、傢俱清單和平面圖中作為傢俱的圖示顯示。最後按一下“完成”。

　　關閉嚮導視窗後，剛才插入的模型就會依據您使用的選項顯示在傢俱目錄或平面圖和傢俱清單中。
　　您可以和對待目錄中預設的傢俱一樣在平面圖中對其進行自訂操作。
　　插入到目錄中的傢俱名稱會顯示為斜體。如要使用傢俱插入嚮導編輯其屬性，請在目錄中按兩下該傢俱，或者將其選中後選擇傢俱→自訂... 。

•

將 3D 模型檔拖放到 Sweet Home 3D 視窗中插入傢俱的方法在 Linux 系統上可能會無法使用。

•

物價目錄不支援在同一類別中存在兩個同名的傢俱。如果您想插入目錄中的傢俱名稱在相應的類別中存在重名現象，您將無法使用傢俱插入嚮導第三步的“繼續”按鈕。

•

您可以直接編輯“目錄類別”下拉式列示方塊的文字區域，用來創建新的類別。

•

將 3D 模型插入傢俱目錄或平面圖取決於您使用傢俱→插入傢俱... 時螢幕焦點的位置。若此時焦點位於傢俱列表、平面圖或 3D 視圖，那麼僅當您在傢俱插入嚮導的第三步中勾選“增加到目錄”後，您倒入的模型才會在傢俱目錄中出現。

•

傢俱插入嚮導中的預設單位取決於作業系統的區域設置，但您仍然可在“參數設置”中將其更改為釐米、米、毫米或英寸。

•

由於所有開啟的 Sweet Home 3D 視窗全都共用一個傢俱目錄，故任何針對其進行的操作（如插入、更改和刪除）均無法被撤銷。

•

插入目錄中的傢俱的屬性會被自動儲存，傢俱的 3D 模型和圖示的檔會被儲存在 Sweet Home 3D 專用的資料夾中。

Sweet Home 3D 說明
     Sweet Home 3D 使用者畫面
 

　　每當您建立或開啟一個家居模型時，Sweet Home 3D 就會開啟一個視窗，您可以在其中對家居模型進行編輯。整個視窗區域被劃分成了四個可調整大小的窗格，如圖所示：

1.

傢俱目錄。
　　該目錄被分為了數個類別，包含了所有您可以增加到家居模型中的傢俱。如要展開某一個類別，請按一下類別左側的加號或三角形，或者按兩下類別條目。

2.

家居傢俱列表。
　　該清單列出了您的家居模型中已使用傢俱的清單，其中預設顯示了傢俱的名稱、大小以及其它的一些屬性。該列表可通過按一下相應的列標題對其進行排序，也可使用傢俱→排序依據功能表項目執行該操作。此外，列表中各個縱列的可見性可以使用傢俱→可見列功能表項目進行設置。

3.

家居平面圖。
　　該窗格中顯示的是您的家居模型的俯視平面圖。您對家居模型所做的所有操作，如繪製牆體和房間、擺放各種傢俱等，都需要在這裡完成。您可以使用滑鼠拖曳或使用鍵盤上的方向鍵調整傢俱的擺放位置。

4.

家居 3D 視圖。
　　該窗格中呈現的是當前家居模型的立體效果圖。您可以使用兩種不同的方式察看您的家居模型。如果您想從一個較高的位置查看家居模型的全貌，請使用“鳥瞰模式”；如果您希望從一個虛擬參觀者的視角細窺家居模型的設計，請使用“虛擬參觀模式”。

　　每一個窗格都可以捕獲螢幕焦點。焦點變化後，Sweet Home 3D 的某些功能表項目所執行的操作也會相應地變化。窗格捕獲焦點後會在其四周顯示出一條彩色框線。如果想在各個窗格之間切換螢幕焦點，您可以使用 Tab 或 Shift + Tab 快速鍵，也可直接用滑鼠在希望獲取焦點的窗格內按一下。

　　用滑鼠拖曳視窗頂部的工具列可將其停駐於視窗的任意一邊上，或者浮動顯示在螢幕上的任意位置。
